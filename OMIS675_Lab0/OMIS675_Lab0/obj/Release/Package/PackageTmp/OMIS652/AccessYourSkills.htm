Introduction:

In the era of modern world, the necessity to learn and expertise different technologies have become must in order to meet with the accelerating demands. It is quite obvious that all the technologies learnt by a person should be evaluated to analyze one’s knowledge in the particular field of study. Accessyourskills.com is a web app which mainly concentrates on evaluating user skills in a specific field of study and compare the test results with highest existing standards.

Business Description:

Accessyourskills.com concentrates mainly in providing an online examination on different aspects of technological advancements. It attracts users with different skills to evaluate their knowledge and it also provides guidelines to users in improving their existing scores by providing them with advanced information like links to the various publishers’ sites. A user may purchase those advanced information to gain more knowledge. It also encourages publishers to provide their content for some price in the website so that the users who were interested in it will purchase them and thus the business promotes in the real world. A single field of study can be provided by one or more publishers and a particular publisher can publish different books on different technologies. On the other hand, accessyourskills.com holds a centralized repository whose data will be equivalent to the knowledge provided by publishers’ books. Any technological evaluation in the website is divided into different categories based on difficulty so that all the users who were pursuing the lowest difficulty level assessment will be provided for free and the next higher levels will only be accessed for a particular price.

Need for Database:

In the sense of real world there were lot of skills a person or an individual needs to expertise on. More over each skill will have its unique blend of features and properties. Some of them define to their own boundaries and some of them provides interoperability with other technologies and it is a never ending process for the technological advancements. Hence there should be a repository to store this vast information which needs to be accessed during every evaluation. The database for this web app would have tables to store user information, user assessments, list of skills available for evaluation, user results and purchases made by the user(if any). All the tables were broken down to the ground by applying the normalization techniques so that the data discrepancy is avoided.

Business Rules:

1.       User must have to register with the web app.

2.       User can be any person like an individual or an employee or a student.

3.       User should provide valid email during registration.

4.       User can have single session at a time.

5.       User can abort his assessment at any time during the assessment and his attempt will be treated as void.

6.       User cannot take multiple assessments at a same time but can map to multiple skills in a single assessment.

7.       User may or may not purchase the publishers’ books.

8.       A charge will be levied on user for taking any assessment with higher difficulty level other than basic.

9.       All Users data will be stored and can be viewed by him at any time unless during assessment.

10.    A user can attempt an assignment only once but can map to a same assignment more than once.

11.    Any assessment with result scoring more than or equal to 70 percentage will be considered as cleared.
